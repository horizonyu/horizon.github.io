##第二种方法创建数据库

调用相应的方法，如 insert(),update(),delete(),query(),
特点：
    -增删改 有返回值，能判断sql语句是否执行成功，查询不够灵活，不能做多表查询，
    -所以增删改 使用第二种方式，查询使用第一种方式，

//增
ContextValue values = new ContextValue();
values.put("name",bean.name);
values.put("phone",bean.phone);
//table: 表名 nullColumHack： 可以为空，表示添加一个空行 ，values: 添加一行数据的值,返回值代表添加这个新的一行的ID -1 代表添加失败
db.insert(table,nullColumHack,values);
long result = db.insert("info",null,values);

db.close();

if(result ！= -1){
//-1代表添加失败
    return true;
else
    return false;

}
