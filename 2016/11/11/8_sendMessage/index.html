#activity之间的跳转、数据传递、listview的数据显示

此应用实现的功能如下：
1 点击添加按钮，可实现添加联系人的功能；
2 点击消息模板按钮，可实现添加消息模板的功能；

##添加联系人的功能

  1. 为了能够实现在被调用的activity关闭后能将数据返回，需要在启动activity时调用startActivityForResult（）方法，此方法包含两个参数 Intent 和 RequestCode 此部分的代码如下:

    //MainActivity.java
//添加联系人的点击事件
    bt_add.setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(View v) {
            Intent intent = new Intent(getApplicationContext(), ContactActivity.class);

            //requestCode 请求码
            startActivityForResult(intent, 1);
        }
    });

  1. 相应的在ContactActivity中，由于联系人信息是由listview显示出来，所以需要涉及到listview适配器的设置，代码如下,其中在contactAdapter中主要用到的是getCount()和getView（）方法：

    //ContactActivity.java
//listview适配器的设置
lv_contact.setAdapter(new contactAdapter());
......
 private class contactAdapter extends BaseAdapter{

    }

}

3.接下来实现的是当点击listview的条目时，将条目上的信息返回，并关闭此activity

    //ContactActivity.java
    private ListView lv_contact;
       private List<Contact> arrayListContact;
    ......
    lv_contact.setOnItemClickListener(new AdapterView.OnItemClickListener() {
    @Override
    public void onItemClick(AdapterView<?> parent, View view, int position, long id) {

        //contact对象是保存在List集合中，所以可以通过集合获取contact对象
        //取出点中条目的数据
        String phone = arrayListContact.get(position).getPhone();

        //把取出的数据返回
        Intent intent = new Intent();
        intent.putExtra("name",phone);

        setResult(10,intent);  //把数据返回给调用者

        //☆☆☆☆☆关闭当前activity，此时Mainactivity的onActivityResult方法就会执行
        finish();

    }
});

4.在MainActivity中处理由activity返回数据的代码如下：

//当开启的activity关闭的时候调用此方法
//此方法用来处理所启动的activity所返回的数据
@Override
protected void onActivityResult(int requestCode, int resultCode, Intent data) {
    super.onActivityResult(requestCode, resultCode, data);

    switch (resultCode){
        //10：表明是contactActivity返回的数据
        case 10:
            String phone = data.getStringExtra("name");
            et_phone.setText(phone);
            break;
        //20：表明是SmsTemplateActivity返回的数据
        case 20:
            String content = data.getStringExtra("content");
            et_content.setText(content);

        }
    }

5.同样在消息模板的代码中，也是遵循同样的步骤,代码如下：

//SmsTemplateActivity.java
public class SmsTemplateActivity extends Activity {

private String objects[] = {"我正在开会，稍后联系", "我正在上课，稍后联系", "我正在开车，稍后联系", "我正在吃饭，稍后联系", "我正在休假，稍后联系"};

@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_template);

    ListView lv_template = (ListView) findViewById(R.id.lv_template);

    //为listview设置适配器
    lv_template.setAdapter(new ArrayAdapter<String>(getApplicationContext(), R.layout.activty_template_item, objects));

    lv_template.setOnItemClickListener(new AdapterView.OnItemClickListener() {
        //当listview的条目被点击时，将条目上的信息返回到调用此activity的调用者

        @Override
        public void onItemClick(AdapterView<?> parent, View view, int position, long id) {
            String content = objects[position];

            Intent intent = new Intent(getApplicationContext(), MainActivity.class);
            intent.putExtra("content",content);

            //☆☆☆☆☆此方法中有两个参数 ResultCode 和 Intent,resultcode可用于区分是由哪一个activity返回的数据，由于在MainActivity中需要得到此类的返回数据，所以必须设置此方法
            setResult(20,intent);

            //处理点击事件后，将此activity关闭
            finish();
        }
    });

    }
}

1）最后是消息的发送处理方法，首先需要在`manifest`中添加发送信息的权限：

    <uses-permission android:name="android.permission.SEND_SMS"/>
2）接着需要调用发送消息的API,代码如下：

    //发送消息的点击事件
    bt_send.setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(View v) {
            String number = et_phone.getText().toString().trim();
            String context = et_content.getText().toString().trim();

            //调用发送消息的API
            SmsManager smsManager = SmsManager.getDefault();

            //消息的分条发送，目的就是防止因为信息过长而无法一次发送，此处就是将较长的信息分成多条发送
            ArrayList<String> strings = smsManager.divideMessage(context);
            for (String str : strings)
            {
                smsManager.sendTextMessage(number,null,str,null,null);
            }
        }

    });
