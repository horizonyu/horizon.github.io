##1. 圆角矩形的使用
1）首先设置圆角矩形的属性

<?xml version="1.0" encoding="utf-8"?>
<shape
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:shape="rectangle">
    <gradient
        android:startColor="#ffffff"
        android:endColor="#ffffff"
        android:angle="45"/>
    <!--设置填充的颜色-->
    <solid android:color="@color/colorGreen"/>
    <!--设置圆角的半径-->
    <corners android:radius="5dp"/>

    <!--设置文字与控件的距离-->
    <padding
        android:left="5dp"
        android:right="5dp"
        android:top="5dp"
        android:bottom="5dp"/>
</shape>

2）其次设置一个选择器，将此图片作为其中的一种状态

<?xml version="1.0" encoding="utf-8"?>
<selector xmlns:android="http://schemas.android.com/apk/res/android">
    <item android:drawable="@drawable/gradient_box" android:state_pressed="true"/>
    <item android:drawable="@android:color/transparent"/>
</selector>

3）将次选择器作为控件的背景

 android:background="@drawable/selector_reset_bc"

##2. sim卡变更报警

SIM卡存储了其唯一性的序列号在本地，如果SIM卡发生变化，其序列号和存储在本地的序列号有差异，此时向预先设置好的安全号码发送短信。

    //1.获取存储在本地的sim卡的序列号
    String localSerialNumber = SpUtil.getString(context, ConstantValues.PHONE_NUMBER, "");

    //2.获取插入爱手机中的sim卡的序列号(TelephonyManager)
    TelephonyManager tm = (TelephonyManager) context.getSystemService(Context.TELEPHONY_SERVICE);
    String simSerialNumber = tm.getSimSerialNumber();

    //3.将两个序列号进行比对
    if(!localSerialNumber.equals(simSerialNumber)){
        //4.如果不相等，则向安全手机发送报警短信
            //获取预先设置的安全号码
        String safeNumber = SpUtil.getString(context, ConstantValues.CONTACT_PHONE, "");
        SmsManager sm = SmsManager.getDefault();
        sm.sendTextMessage(safeNumber,null,"sim changed! pay attetion!",null,null);
    }

    //5.同时不能忘记在Manifest文件所做的变化：
        //设置广播接收器
         <receiver android:name="com.example.horizon.a23_360safe_receiver.a23_360safe_Receiver">
        <intent-filter>
            <action android:name="android.intent.action.BOOT_COMPLETED"/>
        </intent-filter>
          </receiver>

        //增加权限
         <uses-permission android:name="android.permission.RECEIVE_BOOT_COMPLETED"/>
            <uses-permission android:name="android.permission.SEND_SMS"/>
