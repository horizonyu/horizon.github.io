#_ListView的使用

ListView 是一个控件，一个在垂直列表中显示条目的控件，这些条目的内容来自ListAdapter，（EditText , Button TextView......）

1.布局添加 listview 
2.找到listview
3.创建 一个Adapter 适配器，封装4个方法，
4.创建adapter 对象，设置给listview

##listview 的优化
adapter 中 getView()方法，会传进一个 convertview 对象，convertview 是指曾经使用过的view 对象

@Override
   public View getView(int position, View convertView, ViewGroup parent) {

       TextView view = null;
           //convertview 缓存 view,即已经创建的但并没有在屏幕上显示的 view,创建的view数量仅为屏幕能够显示的view数量+1，减少资源的浪费，
           //使用前需要判断，listview 的优化
           if(convertView != null)
           {
               view = (TextView) convertView;
           }
           else
           {
               view = new TextView(mContext);
           }
           view.setText("position:" + position );
           view.setTextSize(100);
           map.put(view.hashCode(),1);

       System.out.println("创建了"+map.size()+"个TextView 对象");
           return view;

##listview 显示原理
1.要考虑listview 显示的条目数 getCount
2.考虑listview 每个条目显示的内容 getView
3.考虑每个item的高度,屏幕的多样化
4.考虑listview的滑动，监听一个旧的条目的消失，新的条目的出现

##复杂listview界面显示，新闻（**重要*）、

2.找到listview

3.获取新闻数据，封装到list集合，作为adapter的数据源，考虑怎么将获取的数据给adapter

4.创建adapter 继承BaseAdapter ,并且实现4个方法
    getCount（）：返回新闻的条数

新闻客户端案例项目和产品

4. 创建一个adapter 集成BaseAdapter 封装四个方法，接收获取的新闻数据
5. 将adapter设置给listview 。
