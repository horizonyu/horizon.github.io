##1. 计算机表示图形的几种方式：

###图形的大小 = 图片的总像素 每个像素的大小
如：800400像素的图片 320000/8 = 40000 byte

####-单色： （每个像素最多可以表示两种颜色），只需要使用长度为1的二进制位来表示，每个像素占1/8byte；

####-16色 ：（每个像素最多可以表示16种颜色） 0000 - 1111 只需要使用长度为4的位二进制表示 那么每个像素占1/2 byte ;

####-256色： （每个像素最多可以表示256种颜色） 0000 0000 - 1111 1111 只需要使用长度为8的位二进制表示 ,每个像素占1 byte ;

####-24位：rgb r 1byte, g 1byte, b 1byte 一个 像素占3个byte

####-在andoroid 下 采用ARGB A 表示透明度， 一个像素表示4byte

####-图片：24003200 ， 手机 320480 ，宽缩放 ：7，高缩放：6，所以按照宽缩放进行，

所以需要想办法获取需要 缩放的图片的像素 以及 手机的像素
