#SharedPreference基础知识

##1、向 SharedPreference 中插入数据

###1)、通过 Context 对象创建一个SharedPreferenceduixang 对象:
//name:shared..的名称，mode: 文件的操作模式
SharedPreference sharedPreference = Context.getSharedpreferences(name,mode);

###2)、得到一个Editor对象:
Editor editor = sharedPreference.editor();

###3)、向 Editor 中添加数据:
//向editor 里面以 key-value 的形式存放数据
editor.putString(key,value);

###4)、提交 Editor 对象 :
//最后就是通过 commit() 方法提交
editor.commit();

##2、从 SharedPreference 中取出数据：

###1)、通过 Context 对象创建一个SharedPreferenceduixang 对象:
//name:shared..的名称，mode: 文件的操作模式
SharedPreference sharedPreference = Context.getSharedpreferences(name,mode);

###2)、使用 SharedPreference 的 getString() 方法获取数据
//key 代表存入的键，deValue 代表默认的值，即当 key 不存在的时候使用此值
String userName = sharedPreference.getString(key,deValue);
Stirng passsWord = sharedPreference.getString(key,deValue);

##sharedPreference 是通过xml文件来作为数据存储；（—-用来存放一些标记性数据，或是一些设置信息—-典型的案例就是软件的欢迎界面，第一次安装软件时，会进入到欢迎界面，但以后再次进入时就不再出现，就是因为在第一次进入到软件时会更改一些变量，用于标记是否是第一次使用。所以如果在手机的设置中点击清除软件的数据和缓存后，再次进入页面，就会出现欢迎界面。。。。。。）
