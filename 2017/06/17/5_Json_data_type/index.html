##2 Handler的使用 （ANR异常）

1.耗时的操作不能放在主线程中，必须放到子线程。
2.子线程不能够更新UI
3.使用Handler解决子线程不能更新UI的问题：
    1.主线程创建一个Handler ，重写handleMessage方法
    2.在子线程中，创建一个Message对象，封装获取的结果，使用主线程中的Handler对象，将Message对象发送到主线程，
    4.主线程中的handlerMessage方法接收子线程传来的message对象，并取到结果，处理结果。
